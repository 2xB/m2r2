Include Markdown file¶

To include markdown file:

.. mdinclude:: path-to-file.md

To include markdown file with specific lines:

.. mdinclude:: included.md
   :start-line: 2
   :end-line: -2

Original included.md file is:

NOT-INCLUDED

#### Include this line

NOT-INCLUDED

#### Include this line
